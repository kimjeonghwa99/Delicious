# Delicious 🍽️

이사·여행·출근 등 환경이 바뀌어도  
현재 위치를 기준으로 주변 생활 편의시설을 빠르게 확인할 수 있는  
**위치 기반 웹 애플리케이션**입니다.

브라우저의 위치 정보를 활용해  
네이버 지도 검색 결과를 새 탭으로 연결합니다.

## 🔗 Demo
https://kimjeonghwa99.github.io/Delicious/

## ✨ 주요 기능
- 현재 위치 자동 인식 (Browser Geolocation API)
- 네이버 지도 검색 연동
- 분류 버튼 제공
  - 편의점
  - 마트
- 환경 변화(이사/외출) 시 별도 설정 없이 자동 적응

## 🛠 사용 기술
- HTML / CSS / JavaScript
- Browser Geolocation API
- Naver Map 검색 URL 연동

## 💡 기획 의도
이사나 새로운 지역으로 이동했을 때  
별도의 설정이나 앱 설치 없이  
내 주변의 기본적인 생활 인프라를 즉시 확인할 수 있도록 기획했습니다.
