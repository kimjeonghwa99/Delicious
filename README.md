# Delicious 🍽️

이사·여행·외출 등으로 환경이 바뀌어도  
**현재 위치를 기준으로 주변 생활 편의시설을 빠르게 확인할 수 있는 위치 기반 웹 애플리케이션**입니다.

브라우저의 위치 정보를 활용하여  
네이버 지도 검색 결과를 **별도 앱 설치 없이 즉시 확인**할 수 있습니다.

## ✨ 주요 기능
- **현재 위치 자동 인식** (Browser Geolocation API)
- **네이버 지도 검색 연동** (검색 결과 새 탭 이동)
- 음식점 / 카페 / 버스정류장 / 지하철역 **빠른 검색 버튼 제공**
- 모바일 환경에 최적화된 **하단 탭 UI**
- 이사·외출 등 환경 변화 시 **별도 설정 없이 자동 적용**

---

## 🛠️ 사용 기술
- HTML / CSS / JavaScript
- Browser Geolocation API
- Naver Map 검색 URL(`/p/search/`) 연동
- GitHub Pages 배포

---

## 💡 기획 의도
새로운 지역으로 이동했을 때  
앱 설치나 복잡한 설정 없이 **내 주변의 기본적인 생활 인프라를 즉시 확인**할 수 있도록 기획했습니다.

특히 모바일 환경에서 한 손으로 쉽게 사용할 수 있도록  
간단한 UI와 최소한의 동작 흐름에 집중했습니다.
